# 🚀 AI-Agent_Mini

AI-Agent_Mini is a lightweight, AI-powered tool designed to dynamically generate code based on user input and save it to a file. This utility streamlines code creation, making it particularly useful for rapid script generation, automation tasks, and prototyping.

## ✨ Features

- 🔹 **Dynamic Code Generation**: Produces code snippets tailored to user specifications.
- 🔹 **File Output**: Automatically saves the generated code to a file for easy access and execution.
- 🔹 **Simplicity and Efficiency**: Offers a straightforward interface for quick automation and prototyping needs.

## 📦 Installation

To use AI-Agent_Mini, follow these steps:

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/Min-Saurabh/AI-Agent_Mini.git
   ```

2. **Navigate to the Project Directory**:

   ```bash
   cd AI-Agent_Mini
   ```

3. **Install the Required Dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

## 🛠 Usage

1. **Run the Main Script**:

   ```bash
   python main.py
   ```

2. **Provide Input**: Follow the on-screen prompts to specify the code you wish to generate.

3. **Access the Generated Code**: The resulting code will be saved to a file in the project directory, ready for use or further modification.
